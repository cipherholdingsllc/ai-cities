# Agentic Leverage

Evaluates value creation efficiency in multi-agent workflows.

## Source model

This model adapts scale economies and agglomeration economies. Urban systems become more productive when specialization and clustering create gains, but those gains can be offset by transportation, coordination, or congestion costs. In AI Cities, agentic leverage measures whether clustered agents, tools, and workflows create verified value after execution, coordination, context, verification, rework, and risk costs are included.

## Model

Agentic leverage measures verified value output relative to total system costs.

AL_H = V^✓_H / (C_exec + C_coord + C_ctx + C_verify + C_rework + C_risk)

Where:
- H: human-agent hybrid system
- V^✓_H: verified value created
- C_exec: execution costs
- C_coord: coordination costs (folded in)
- C_ctx: context allocation costs
- C_verify: verification costs
- C_rework: error correction costs
- C_risk: risk mitigation costs

## Interpretation

AL_H > 1 indicates net value creation. Maximizing leverage requires balancing costs against verified outputs.

## Agentic Application

In AI cities:
- Multi-agent teams outperform solo agents
- Coordination costs offset by parallel processing
- Verification ensures quality

## Healthcare Example

Surgical planning team: human surgeon + AI assistants. Leverage calculation shows when AI augmentation creates more verified outcomes than costs incurred.

## Optimization

- Minimize coordination overhead
- Maximize parallel value streams
- Balance verification with speed

## Relationship to Other Models

Integrates with congestion (resource costs) and behavioral externalities (rework costs).
