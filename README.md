# AI Cities

**Urban economics for agentic AI infrastructure.**

## Thesis

AI systems are becoming environments, not tools; environments need civic architecture.

As AI agents proliferate—from autonomous coding assistants to self-driving fleets—they form interconnected systems that behave like cities: complex, emergent, and requiring governance. This repository provides a math-first framework for designing civic safeguards in agentic AI ecosystems, drawing from urban economics principles.

## Scope

This repository is a public framework, not an implementation or simulator. It offers conceptual models, frameworks, and templates for understanding and mitigating externalities in agentic AI systems. All content is under 500 words per document (except this README), emphasizing mathematical rigor over code.

## Core Models

Five foundational models anchor the framework:

1. **Congestion Externality** - Quantifies resource contention in agentic systems
2. **Behavioral Externality Multiplier** - Measures downstream costs of agent actions
3. **Agentic Leverage** - Evaluates value creation efficiency in multi-agent workflows
4. **Risk-Adjusted Autonomy** - Balances autonomy with safety constraints
5. **Context Allocation** - Optimizes resource distribution in cognitive environments

## Frameworks

- **AI City Subsystems** - Functional decomposition of agentic ecosystems
- **Risk Zoning** - Spatial organization of agent permissions and capabilities
- **Memory Records and Maps** - Persistent knowledge infrastructure
- **Verification and Civic Safeguards** - VIGIL framework for runtime governance

## Applications

- **Healthcare AI City** - Agentic systems in medical decision-making
- **Developer Tools AI City** - AI-assisted software development ecosystems
