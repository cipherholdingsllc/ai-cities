# Sources

## Primary foundation

- Brueckner, J. K. (2011). *Lectures on Urban Economics*. MIT Press.

AI Cities uses Brueckner as its primary urban-economics foundation, especially for congestion, externalities, agglomeration, spatial structure, land scarcity, public goods, pollution, crime, and quality-of-life measurement.

## Concept map

| Urban economics concept | AI Cities translation |
| --- | --- |
| Scale economies | Agent/tool specialization and workflow productivity |
| Agglomeration economies | Gains from clustering agents, tools, memory, data, and workflows |
| Transport costs | Tool-routing, handoff, search, dependency traversal, and coordination costs |
| Urban spatial structure | Context allocation and cognitive centrality |
| Land scarcity | Scarce context, memory, compute, attention, and human review |
| Freeway congestion | Latency, contention, verification load, and coordination burden |
| Public goods | Open benchmarks, documentation, evals, standards, and shared safety conventions |
| Pollution | Hallucinations, slop, corrupted memory, misinformation, and cleanup cost |
| Crime | Prompt injection, fraud, malicious agents, data poisoning, and impersonation |
| Quality of life | Trust, usability, reliability, cognitive burden, and system health |
